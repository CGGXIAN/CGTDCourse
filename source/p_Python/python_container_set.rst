=============================
Python容器：集合 {}
=============================

容器是对专门用来装其它对象的数据类型的统称。在Python中，有四种最常见的内建容器类型：列表（list）、元组（tuple）、字典（dict）、集合（set）。Python语言内部实现细节也与这些容器息息相关。比如Python的类实例属性、全局变量globals()等都是通过字典类型来存储的。

-----------------
集合 set {}
-----------------

集合与字典都是使用花括号{}来定义，区别是字典是通过键值对的方式存储。
