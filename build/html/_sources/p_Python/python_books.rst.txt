==============================
Python相关书籍推荐以及阅读感想
==============================

这里将自己阅读过的一些书和读后感推荐给大家。


- 《流畅的Python》 极好的一本进阶书，建议基础好的同学多看几遍。
- 《Python学习笔记雨痕版》 作者的笔记，很多心得写的不错，建议一看。
- 《Python Cookbook》 进阶的书，需要一定基础，给了现实问题的很多解决方案，值得没事的时候翻阅看看。
- 《廖雪峰Python教程》 适合入门，给了很多案例都是极好的，基础不好可以多看看。
- 《编写高质量代码 改善Python程序的91个建议》 进阶的书，如果你觉得你写的代码不够高档可以茶余饭后阅读，尝试用另一种解决方案。

- 《像计算机科学家一样思考Python》 基础的书，内容比较繁琐，当字典看看。
- 《Python源码剖析》 比较深奥的一本书，涉及到Python底层原理，适合喜欢追根究底的你，要对Python有足够深的理解才能看懂。

- 《Python学习手册》：长达800页的一本书，内容很详细，详细到很啰嗦，知识体系比较杂乱，可以当Python字典在用到的时候查阅相关知识点，不建议从头看到尾。
- 《Python技术手册》：
- 《Python参考手册》：
- 《Python 3程序开发指南》：
- 《Python编程从入门到实践》：
- 《Python核心编程》：长达900页的一本教科书，内容详细允长，可以当Python字典在用到的时候查阅相关知识点，后面有讲到多线程编程，网络编程，以及数据库编程等等。
- 《Python高级编程》：
- 《Effective Python 编写高质量Python代码的59个有效方法》：一本进阶的书，提升代码质量的59条建议，可以茶余饭后阅读，尝试用更高端的解决方案。
- 《可爱的Python》：前面故事篇可能跟我自己实际工作关联不大，没多大意思，但从环境篇，语法篇，模块篇，框架篇，友邻篇倒是讲解了很多有用的知识点，估计很多人放弃在故事篇了，没有看完可以回头再翻翻。
- 《简明Python教程》：内容教科书式的编排，比较入门，没多大意思。
