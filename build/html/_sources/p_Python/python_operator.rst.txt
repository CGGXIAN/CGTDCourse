======= ================================================== =============================
``+``    加 - 两个对象相加                                      2 + 3 输出结果 5
``-``    减 - 得到负数或是一个数减去另一个数                      2 - 3 输出结果 -1
``*``    乘 - 两个数相乘或是返回一个被重复若干次的字符串           2 * 3 输出结果 6
``/``    除 - x除以y                                            3 / 2 输出结果 1
``%``    取模 - 返回除法的余数                                   3 % 2 输出结果 1
``**``   幂 - 返回x的y次幂                                      2 ** 3 输出结果 8
