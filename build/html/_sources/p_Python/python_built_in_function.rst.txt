=============================
Python内置函数
=============================

Python内置函数也叫内建函数（built-in function），内置函数大约有60来个，都非常有用。


* len()

* enumerate()

.. code-block:: python

    for i, item in enumerate(myList):
        print(i, "-->", item)

* int() | float() | str() | bool()

* eval() | exec()

eval()会将一个字符串当成一个表达式来执行，返回表达式执行后的结果。
exec()会将一个字符串当成程序来执行。


* max() | min() | sum()

* abs()
