=============================
Python函数式编程
=============================

* 函数式编程，也可以归到面向过程的程序设计，但思想更接近于数学计算。
* 函数式编程是一种抽象程度很高的编程范式，纯粹的函数式编程语言编写的函数没有变量，因此任意一个函数，只要输入时确定的，输出就是确定的，这种纯函数我们称之为没有副作用。
* 函数式编程的一个特点是，允许把函数本身作为参数传入另一个函数，还允许返回一个函数。
