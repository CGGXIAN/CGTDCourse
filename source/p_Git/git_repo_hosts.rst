=============================
Git代码仓库托管平台有哪些？
=============================

Git托管平台推荐使用Github、Gitlab，国外服务器访问上可能有些丢包延时的问题，也可以使用本地服务器安装虚拟机搭建本地Gitlab以满足大型团队协作。

------------------
国外平台
------------------

* Github

网址：https://github.com/

* Gitlab

网址：https://gitlab.com

* BitBucket

网址：https://bitbucket.org/

* SourceForge

网址：https://sourceforge.net/

------------------
国内平台
------------------

* 码云

网址：https://gitee.com/
